# Папка для картинок штрафов

Поместите сюда картинку, которая будет отправляться при получении штрафов.

## Необходимый файл:

- `penalty.jpg` - ОДНА картинка для ВСЕХ штрафов (1, 2, 3, 4+)

## Требования:

- Формат: JPG или PNG
- Рекомендуемый размер: до 1920x1080

## Примечание:

Эта картинка используется для визуального оформления сообщений о всех штрафах.
Текст сообщения меняется в зависимости от количества штрафов, но картинка одна.
