# Папка для картинок заданий курса

Поместите сюда картинки для всех 14 заданий.

## Формат названий файлов:

- `task_1.jpg` - картинка для задания 1
- `task_2.jpg` - картинка для задания 2
- `task_3.jpg` - картинка для задания 3
- ...
- `task_14.jpg` - картинка для задания 14

## Требования:

- Формат: JPG или PNG
- Рекомендуемый размер: до 1920x1080
- Все файлы должны быть пронумерованы от 1 до 14

## Примечание:

Пути к этим картинкам также можно указать в базе данных в таблице `course_tasks` в поле `image_path`.
