# Финальные сообщения 15 дня

⚠️ **ВАЖНО:** Финальные сообщения отправляются **БЕЗ медиафайлов**.

Эта папка зарезервирована на случай, если в будущем понадобится добавить картинки к финальным сообщениям.

## Настройка финальных сообщений

Финальные сообщения отправляются только текстом (has_media = false).

Для редактирования текстов используйте SQL:

```sql
UPDATE final_messages 
SET message_text = 'Ваш новый текст' 
WHERE message_number = 1;
```

## Время отправки

- **10:00** - Первое финальное сообщение (поздравление с завершением курса)
- **15:00** - Второе финальное сообщение
- **15:55** - Третье финальное сообщение (финал)

Эти сообщения отправляются **всем** пользователям, которые завершили 14 задание, независимо от того, как прошел их курс (с штрафами или без).

## Если нужно добавить картинки

Если в будущем потребуется добавить медиафайлы:

1. Добавьте картинки в эту папку
2. Обновите таблицу final_messages:
   ```sql
   UPDATE final_messages 
   SET has_media = true,
       media_path = 'media/final/final_1.jpg'
   WHERE message_number = 1;
   ```
3. Перезапустите бота
