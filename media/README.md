# Папка для медиафайлов

Поместите сюда следующие файлы:

## Необходимые файлы:

1. **welcome_video.mp4** - Видеокружок для приветствия
   - Формат: MP4
   - Соотношение сторон: 1:1 (квадратное)
   - Рекомендуемый размер: до 1 минуты
   
2. **channel_request.jpg** - Картинка при запросе канала
   - Формат: JPG или PNG
   - Рекомендуемый размер: до 1920x1080
   
3. **final_message.jpg** - Картинка после успешной регистрации
   - Формат: JPG или PNG
   - Рекомендуемый размер: до 1920x1080
   
4. **instruction.mp4** - Видео с инструкцией
   - Формат: MP4
   - Любая длительность

## Примечания:

- Все пути к файлам настраиваются в файле `config.py`
- Если какой-то файл отсутствует, бот пропустит его отправку и продолжит работу
- Убедитесь, что файлы имеют правильные названия или обновите пути в `config.py`
