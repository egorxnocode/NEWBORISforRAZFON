# Папка для картинок напоминаний

Поместите сюда ТРИ РАЗНЫЕ картинки для напоминаний.

## Необходимые файлы:

- `reminder_1.jpg` - картинка для первого напоминания (8:50)
- `reminder_2.jpg` - картинка для второго напоминания (9:20)
- `reminder_3.jpg` - картинка для третьего напоминания (9:35)

## Требования:

- Формат: JPG или PNG
- Рекомендуемый размер: до 1920x1080

## Примечание:

Каждое напоминание отправляется со своей уникальной картинкой.
Это позволяет визуально различать напоминания.
